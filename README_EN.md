# Ethereum Whitepaper (Persian Translation)

This repository contains an **unofficial Persian (Farsi) translation** of the original **Ethereum Whitepaper**, written by Vitalik Buterin in 2014.

The goal of this project is to provide an accurate, readable, and well-formatted Persian version of the Ethereum Whitepaper while preserving the original structure, sections, equations, and overall layout of the official PDF.
